# Mi Web

Este es un proyecto de ejemplo de una página web simple hecha con HTML, CSS y JavaScript.

## Descripción

Esta web está diseñada como una página de demostración. Contiene:

- `index.html`: página principal
- `style.css`: estilos de la página
- `script.js`: funcionalidad JavaScript (opcional)
- Carpeta `img/` para imágenes (opcional)

## Cómo ver la web

Puedes ver la web de dos maneras:

1. **Localmente**  
   - Clona el repositorio:  
     ```bash
     git clone https://github.com/TU_USUARIO/TU_REPOSITORIO.git
     ```  
   - Abre `index.html` en tu navegador.

2. **En línea con GitHub Pages**  
   - Ve a **Settings → Pages** en tu repositorio.  
   - Selecciona la rama `main` y la carpeta `/ (root)`.  
   - GitHub generará un enlace para acceder a la web.
