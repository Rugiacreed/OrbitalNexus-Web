# Mi Web

Este es un proyecto de ejemplo de una página web simple hecha con HTML, CSS y JavaScript.

## Descripción

Esta web está diseñada como una página de demostración. Contiene:

- `index.html`: página principal
- `style.css`: estilos de la página (en desarrollo)
- Carpeta `assets/` para imágenes

## Cómo ver la web

Puedes ver la web en orbitalnexusassessors.com.
